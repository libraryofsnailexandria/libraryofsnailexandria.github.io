My simple tips sourced from other creators with my own twist

    Personality > character traits, flaws, and desires
    Scenario > Backstory fuelled and driven by desires of personality, then what is happening in the present is the basis for the Plot.
    Description > Write a summary of where the character is post their Backstory and thus what the Plot hook is
    First Message > Write the first interaction that compels the character and user into the Plot hook
    Example Conversation > an extension of the First Message with hypotheticals written in that help direct to the intended next step of the Plot
